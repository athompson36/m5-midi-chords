QUICK START
===============================================================

DE
1. Python installieren: https://www.python.org/downloads/
2. M5Stack CoreS3 per USB-C Datenkabel verbinden (Datenleitung, nicht nur Laden).
3. Windows: flash_firmware.bat doppelklicken.
4. macOS/Linux: python3 flash_firmware.py ausfuehren.

EN
1. Install Python 3: https://www.python.org/downloads/
2. Connect M5Stack CoreS3 with a USB-C cable that supports data.
3. Windows: double-click flash_firmware.bat.
4. macOS/Linux: run python3 flash_firmware.py.

===============================================================
AFTER FLASH: EXPECTED BEHAVIOR (CoreS3 UI)

- Display shows title "M5 CoreS3 Chord Suggester".
- Bottom bar: BACK (left), SELECT (center), FWD (right) for chord list navigation.
- Short taps on BACK/FWD move selection; SELECT confirms or steps through settings rows.
- Settings: on the main screen, place two fingers on BACK and FWD at once and hold ~0.8 s.
- In settings, BACK/FWD move the highlighted row; SELECT changes the value (last row saves and exits).
- Adjustable: MIDI out channel, MIDI in (OMNI or channel), brightness, output velocity.
- See README.md for build, tests, and PlatformIO upload.

===============================================================
IF SOMETHING FAILS

- Port not found:
  Try download mode: hold BOOT, press RST, release BOOT; or try another USB port/cable.
- Flashing fails:
  Run python flash_firmware.py manually, or use FLASH_INSTRUCTIONS.md.
- Still blocked:
  Open FLASH_INSTRUCTIONS.md for full steps.
